# SVG Logo Set

This folder contains a working set of SVG logos for AI labs and common training/content sources.

Files:

- `openai.svg`: Wikimedia Commons redirect for `OpenAI logo 2025 (symbol).svg`
- `anthropic.svg`: copied from `simple-icons`
- `xai.svg`: Wikimedia Commons redirect for `XAI Logo.svg`
- `mistral-ai.svg`: copied from `simple-icons`
- `deepseek.svg`: Wikimedia Commons redirect for `DeepSeek logo.svg`
- `qwen.svg`: Wikimedia Commons redirect for `Qwen logo.svg`
- `z-ai.svg`: official `z.ai` site asset at `https://z-cdn.chatglm.cn/z-ai/static/logo.svg`
- `moonshot-ai.svg`: traced from the official Moonshot AI PNG wordmark
- `moonshot-ai.png`: original Moonshot AI PNG source used for tracing
- `wikipedia.svg`: copied from `simple-icons`
- `reddit.svg`: copied from `simple-icons`
- `arxiv.svg`: copied from `simple-icons`
- `new-york-times.svg`: copied from `simple-icons`
- `the-washington-post.svg`: copied from `simple-icons`
- `github.svg`: copied from `simple-icons`
- `stack-exchange.svg`: copied from `simple-icons`
- `hugging-face.svg`: copied from `simple-icons`

Notes:

- `moonshot-ai.svg` is the only non-direct source SVG in this folder. I generated it locally from Moonshot's official PNG because I couldn't find a public SVG asset from them in the usual logo repositories.
- If you want, I can do one more cleanup pass next and normalize all of these to a shared viewBox, transparent background, and consistent filename/style conventions for compositing.
