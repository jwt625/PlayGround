## Insights

### Browsing Patterns
- Heavy usage of X (Twitter) - nearly half of all tabs
- Significant research activity (Google Scholar, Nature, arXiv, IEEE)
- Development work (GitHub, Stack Overflow)
- Video consumption (YouTube)

### Temporal Patterns
- Full year of data from November 2024 to November 2025
- Tabs are typically short-lived (median < 1 hour)


### Data Quality
- All 30 JSON files successfully processed
- Both old format (without metadata) and new format (with metadata) handled
- All tree branches traversed and analyzed
- Individual node timestamps utilized for temporal analysis
