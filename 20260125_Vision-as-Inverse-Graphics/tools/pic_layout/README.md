# PIC Layout Tools

Core tools for PIC layout generation workflow.

## Files

- `manual_test.py` - Execute generator code, render to PNG, save outputs
- `call_verifier.py` - Call Qwen3-VL verifier API with target/current comparison
- `analyze_target.py` - Initial target analysis before starting iterations
- `run_iteration.py` - Helper for running single iteration (chip1 example)
