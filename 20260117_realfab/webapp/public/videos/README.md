# Video Assets

## Placeholder Video

The `announcement-placeholder.mp4` file should be placed here.

For now, the video player will show a placeholder message if the video file is not available.

## Video Specifications

- Format: MP4 (H.264 codec recommended)
- Aspect Ratio: 16:9
- Recommended Resolution: 1920x1080 or 1280x720
- File Size: Keep under 50MB for web performance

## Usage

The video is used in the Hero section's VideoModal component.
