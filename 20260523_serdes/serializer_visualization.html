Pure 16:1 NRZ Serializer: mux tree, clocks, retiming, edge regeneration

16 bits @ 7 GHz word rate -> 112 Gb/s serial stream. UI = 8.93 ps. Word period = 16 UI = 143 ps.

1. Serializer Mux Tree / Trace Evolution

All streams are shown. Earlier stages are arranged so the last 2:1 mux receives even-position and odd-position streams, producing the final lane order b0, b1, ... b15 within each word.

2. Final 2:1 Interleaving Zoom

The final mux is controlled by 56 GHz half-rate timing: each input bit lasts 2 UI, and the output alternates every UI.

3. Latch Edge Regeneration Zoom

Conceptual model only: a slow/degraded input transition is sampled and relaunched as a sharper output transition aligned to a clean clock edge.

4. Latch -> 2:1 Mux -> Latch Panel

Logical 2:1 muxing is surrounded by active sampling and relaunch stages. The fast final stages may be custom differential high-speed logic.
