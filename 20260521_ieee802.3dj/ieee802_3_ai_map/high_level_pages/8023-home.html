IEEE 802.3 ETHERNET WORKING GROUP
  • A requests for agenda time for a PLCA Multidrop Latency, Jitter, and Scalability Enhancements call for interest has been received.
    It will be considered at the July 2026 IEEE 802.3 Plenary session.
  • The IEEE 802.3 Working Group develops standards for Ethernet networks. We have a number of active projects, study groups, and ad hocs as listed below:
    • IEEE P802.3dg 100 Mb/s Long-Reach Single Pair Ethernet Task Force.
    • IEEE P802.3dj 200 Gb/s, 400 Gb/s, 800 Gb/s, and 1.6 Tb/s Ethernet Task Force.
    • IEEE P802.3dk Greater than 50 Gb/s Bidirectional Optical Access PHYs Task Force.
    • IEEE P802.3dm Asymmetrical Electrical Automotive Ethernet Task Force.
    • IEEE P802.3dp Cabling Restrictions for Single Pair Power over Ethernet Task Force.
    • IEEE Std 802.3-2022/Cor 2 (IEEE 802.3dr) Optical Automotive Ethernet TDFOM Task Force.
    • IEEE P802.3dq Pin-Optimized PHY Interface Task Force.
    • IEEE P802.3ds 200 Gb/s per Wavelength MMF PHYs Task Force.
    • IEEE P802.3dt Ethernet Metadata Services Task Force.
    • IEEE P802.3 (IEEE 802.3du) Revision to IEEE Std 802.3-2022 Maintenance #18 Task Force
    • IEEE 802.3 400 Gb/s/Lane Signaling Study Group.
    • IEEE 802.3 FMP Ethernet Interoperability Study Group.
    • IEEE 802.3 New Ethernet Applications Ad Hoc.
    • IEEE 802.3 New Ethernet Applications Ad Hoc Ethernet for AI Assessment
    • IEEE 802.3 PDCC Ad Hoc.
    • IEEE 802.3 Channel Operating Margin (COM) Open Source Project Ad Hoc.
    • IEEE 802.3 YANG Open Source Project Ad Hoc.
