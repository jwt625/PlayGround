# SLM-Guessr

Training tool for developing intuition about SLM phase masks and 2D FFT behavior.

## Development

```sh
# Install dependencies
pnpm install

# Start dev server
pnpm run dev

# Build for production
pnpm run build

# Preview production build
pnpm run preview
```

## Project Structure

```
src/
  lib/
    components/       # Reusable UI components
      Header.svelte
      GifPlayer.svelte
      SampleCard.svelte
    stores/           # State management
    types.ts          # TypeScript definitions
  routes/
    +page.svelte      # Home
    gallery/          # Gallery mode
    quiz/             # Quiz mode
static/
  assets/             # Generated GIF samples (from Python)
```

## Generating Samples

Training samples are generated by the Python scripts in the parent directory.
See `slm_guessr/` Python package for generation.

Generated assets go into `static/assets/` with a `samples.json` manifest.

## Deployment

Configured for GitHub Pages via static adapter.
Build output goes to `build/` directory.
